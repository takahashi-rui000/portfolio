RPG風ゲーム（Java Swing）

JavaのSwingを使って作成した、GUIのRPG風ゲームです。

専門学校一年目の最後に取り組んだ課題で、Swingの知識が全くない状態から一人で、独学で学びながら開発を進めました。

手探りで始めた開発だったこともあり、設計面での課題は残りましたが、GUI開発やイベント処理基礎を実践的に習得するいい経験となりました。


このゲームはターン制のRPG風ゲームです。

フィールドは二次元配列で管理されており、プレイヤーの移動先の数値に応じて町、カジノ、敵とのエンカウント、ボス戦などの処理が行われます。

敵との戦闘中は攻撃と回復、逃げるという行動が可能で、攻撃時にミニゲームが発生し、攻撃の威力が増減します。

敵との戦闘に勝利すると経験値・ゴールドを取得できます。

町では装備や回復アイテムを買うことができます。

jsonファイルへの書き出しによってセーブ機能を実現しました。


使用技術：Java, Swing, Jackson

Git Hub：Gitへ

タイピングゲーム（JavaScript）

JavaScriptで作成したブラウザ上で動作するタイピングゲームです。

発案から開発まで一人で行いました。

基本的なタイピングゲームの要素に加えて、落ちてくる数字をタイミングよく入力する必要のあるゲームです。

タイトルページから難易度の調整と遊び方の閲覧をできます。

通常の入力フォームを使用せず、イベントリスナーからキー入力を取得し、日本語に変換する独自ロジックをJavaScriptで実装しました。


使用技術：HTML, CSS, JavaScript

公開URL：タイピングゲーム

ユーザー管理システム（JSP/Servlet）

専門学校の課題として作成した、基本的なユーザー管理機能を備えたWebアプリケーションです。

ユーザー登録、ログイン認証、プロフィール編集、ユーザー情報の削除が可能です。

バックエンドはJSP/Servletを用い、MariaDBをデータベースとして利用しています。

セッション管理によるログイン状態の維持や、DAOを用いたデータベース操作の分離を実装しています。

また、重複している処理を一つのJSPファイルにまとめることで、コードの再利用性を高め、保守性の向上に努めました。

この課題を通じて、Webアプリケーションの基礎的な仕組みや、バックエンドとデータベース連携の経験を得ました。


使用技術：Java (JSP/Servlet), MariaDB, HTML

Git Hub：Gitへ
